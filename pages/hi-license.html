Step-by-Step

Register, Generate, Claim, Verify

Follow this order when testing the full HI License flow on a fresh browser.

  1. 01
    Register local access

    Open HI App, choose Setup on the login page, create a local username and password, then enter the protected workspace.

    Open setup
  2. 02
    Generate your HDI

    Use Edit Identity in HI App. Save name, role, location, and mission. HI generates your Human Digital Identity automatically.

  3. 03
    Create or select content

    Add a sample note, goal, project, or event in HI App, or use the manual Claim Content form below.

  4. 04
    Claim and generate license

    Choose content type and license type, add a description, then generate the certificate. Use sample data only while testing.

  5. 05
    Verify and export

    Open the certificate, check HDI, license ID, hash, and timestamp. Export a JSON backup when the flow works.
